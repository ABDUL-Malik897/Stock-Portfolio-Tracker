# Stock Portfolio Tracker
A simple stock tracker that calculates total investment based on manually defined stock prices. 



---



```
A beginner-friendly Python project that calculates total investment value using manually defined stock prices.

## Features
- User enters stock names and quantity
- Uses predefined stock prices
- Calculates total investment amount
- Saves portfolio summary into a text file
- Easy to understand and modify

## Technologies Used
- Python

## Concepts Used
- Dictionary
- Loops
- Input/Output
- File Handling
- Basic Arithmetic

## Available Stocks
Example stocks included in the project:

- AAPL
- TSLA
- GOOG
- MSFT
- AMZN

## How to Run

1. Save the file as `portfolio_tracker.py`
2. Open terminal or command prompt
3. Run:

```bash
python portfolio_tracker.py
```
